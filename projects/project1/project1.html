CIRCLE - Tech karo

The #TechKaro program is one of CIRCLE's key initaitives, run in partnership with EngroVopak & Engro Foundation. The project is currently
training nearly 300 young people, primarily women, from underserved communities in tech-besed courses such as Front-end
Web Development (HTML, CSS, Java Script, React), Graphic Design, and Digital Marketing.
